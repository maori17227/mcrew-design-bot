# Налаштування Telegram авторизації для сайту

## Проблема
Telegram авторизація не працює на сайті через помилку "Bot domain invalid"

## Рішення

### Крок 1: Додати домен в BotFather

1. Відкрийте Telegram і знайдіть [@BotFather](https://t.me/BotFather)

2. Відправте команду:
```
/setdomain
```

3. Виберіть вашого бота: `@mcrew_bot`

4. Введіть домен вашого сайту:
```
maori17227.github.io
```

5. BotFather підтвердить, що домен додано

### Крок 2: Додати localhost для тестування (опціонально)

Якщо хочете тестувати авторизацію локально:

1. Знову відправте `/setdomain` в BotFather
2. Виберіть `@mcrew_bot`
3. Введіть:
```
localhost
```

**ВАЖЛИВО:** Telegram дозволяє тільки ОДИН домен на бота. Тому для production використовуйте `maori17227.github.io`, а для тестування можете тимчасово змінити на `localhost`.

### Крок 3: Перевірка

1. Відкрийте сайт: https://maori17227.github.io/mcrew-design-bot/
2. Натисніть кнопку "Login" в навігації
3. Повинен з'явитися віджет Telegram авторизації
4. Натисніть "Login with Telegram"
5. Авторизуйтесь через Telegram

## URL сайту

- **Production:** https://maori17227.github.io/mcrew-design-bot/
- **Local:** http://localhost:8000 (коли запущено локальний сервер)
