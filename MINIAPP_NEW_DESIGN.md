# 🎨 MↃREW Mini App - Новый Дизайн

## Что нового:

### ✨ Минималистичный дизайн
- Стиль как на скриншоте: чистый, минималистичный
- Крупная типографика (72px логотип)
- Список вместо карточек
- Тонкие разделители между элементами

### 🌓 Светлая и темная тема
- **Светлая тема** (по умолчанию):
  - Фон: #f5f5f5 (светло-серый)
  - Карточки: #ffffff (белый)
  - Текст: #000000 (черный)
  
- **Темная тема**:
  - Фон: #0a0a0a (черный)
  - Карточки: #151515 (темно-серый)
  - Текст: #ffffff (белый)

- Кнопка переключения темы (🌙/☀️) справа вверху
- Автоматическое определение темы из Telegram
- Плавные переходы между темами

### 🎯 Анимации
- Slide-in анимация при открытии экранов
- Плавное движение элементов при нажатии
- Fade-in для логотипа
- Smooth transitions для всех элементов

### 📱 Улучшенный UX
- Крупные кликабельные области
- Визуальная обратная связь при нажатии
- Тени и hover эффекты
- Адаптивный дизайн
