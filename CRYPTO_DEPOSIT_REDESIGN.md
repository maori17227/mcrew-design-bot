# План переделки системы пополнения

## Проблемы:
1. Все методы на одной странице - выглядит ужасно
2. TON Connect не работает на обычном сайте
3. Нет нормальных инструкций для USDT

## Решение:

### Для обычного сайта (website):
- Убрать TON Connect полностью
- Оставить только USDT (TRC-20)
- Показать адрес + QR код
- Простая инструкция

### Для Mini App (miniapp):
- TON Connect (работает только в Telegram)
- USDT (TRC-20) с адресом
- Вкладки для переключения

## Структура модального окна:

```
┌─────────────────────────────────┐
│  Deposit Cryptocurrency    [X]  │
├─────────────────────────────────┤
│                                 │
│  [TON]  [USDT]                 │  <- Вкладки
│                                 │
│  ┌───────────────────────────┐ │
│  │  Контент выбранной вкладки │ │
│  │                            │ │
│  │  - Адрес                   │ │

│                                 │
└─────────────────────────────────┘
```
